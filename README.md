# SLG 战斗模拟器

基于 v1.4 战斗公式的自动战斗游戏原型。

## 快速开始

```bash
cd slg-battle
npm install --ignore-scripts
npm run dev
```

浏览器打开 `http://localhost:5173`，选择你的阵容开始战斗。

## 战斗规则（v1.4）

### 棋盘
- 3×3 九宫格，分为前/中/后三排
- 每方最多 8 个角色上场

### 战斗流程
- 自动战斗（非玩家操作）
- 基于时间刻度的帧更新（100ms/tick）
- 2x 速度播放
- 各单位按攻击间隔自动普攻
- 冷却到则释放技能

### 伤害公式
```
防御减伤 = min(def / (def + 200), 80%)
前排减伤 = 0.9（前排单位）
格挡概率 = 兽族/战士35% 猎人/法师25% 亡灵/龙族15%
格挡值 = 兽族 floor(def/5) 其余 floor(def/7)
纯粹伤害 = 不计算防御/格挡，受前排/其他减伤影响
暴击 = 基础5%，暴击伤害135%
```

### 种族羁绊（简化版，3人=满级）
| 种族 | 效果 |
|------|------|
| 兽族 | 攻+25%，防+15%，半血攻击吸血20% |
| 猎人 | 攻+40%，被切入100%无视防御反击 |
| 战士 | HP+50%，每10s免异5s |
| 法师 | 敌方魔防-60%，技能30%减CD10% |
| 亡灵 | 攻+20%，防+25%，首个阵亡7s后复活35%HP+无敌1s |
| 龙族 | 攻+30%，技能起始冷却缩短50% |
